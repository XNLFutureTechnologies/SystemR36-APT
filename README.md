# SystemR36-APT
This is the main UPDATE REPO for SystemR36, this repo is used by SystemR36 to get it's updates directly from online, and it can also be used by SystemR36 users to download packages manually to install them via the offline method. 

# WARNING
DO NOT use this repo or the (possible) packages in it as of yet! These are NOT done yet, and are NOT intended to be used as it at the moment. SOME packages MIGHT be production ready already, and are published here for those using and/or testing the Alpha/Beta test installation of SystemR36, but these files are in NO WAY intended for public use yet. Also keep in mind that these packages are NOT meant to be used on (or even suitable for) (d)ArkOS in ANY WAY! They often depend on additional files, services and configuration from SystemR36.
